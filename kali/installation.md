# Download and Installation of Kali
If you are very beginner in cybersecurity, first learn how to download Kali Linux.
It is absolutely for beginners because there are many other OS are also available for hacking and practice.

SO, firstly, you need to go to the Kali official website: https://www.kali.org/
where you can find the ISO file of Kali.

--> Click on the Download button 
--> Now you can see different files, but we need to choose the virtual machine
--> Now you see a different machine for a different virtual environment 
    Now the question is which file I should download 

--> Before that, we have to download a VMware Workstation or VirtualBox.I added both site links; just download it on your machine according to your preferences. 
    VMWare Workstation: https://www.vmware.com/products/desktop-hypervisor/workstation-and-fusion
    Virtualbox: https://www.virtualbox.org/wiki/Downloads

  --> After downloading your virtual environment according to that, choose your Kali machine shown in "https://www.kali.org/get-kali/#kali-virtual-machines" on this site.
  --> After downloading, just open the folder of Kali and double-click on the VMDK file; it will automatically open in the virtual environment.
