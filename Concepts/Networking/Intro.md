# Networking in Cybersecurity
Networking in cybersecurity means how computers connect and communicate with each other securely. It focuses on protecting data while it is being transferred between systems.

## Why Networking is Important in Cybersecurity
Helps in secure communication between devices
Prevents unauthorized access
Protects sensitive data from attackers
Ensures data integrity and availability

## Basic Concepts
1. Network
A network is a group of connected devices (computers, servers, mobiles) that share data.

2. IP Address
A unique address assigned to each device on a network (like a home address).

3. Protocols
Rules that define how data is transmitted.
Examples:
HTTP / HTTPS
FTP
TCP / IP

4. Firewall
A security system that filters incoming and outgoing traffic.

5. Router
A device that connects different networks and forwards data.

## Common Network Attacks
Man-in-the-Middle (MITM) – attacker intercepts communication
DDoS (Distributed Denial of Service) – overloads a server
Packet Sniffing – capturing data packets
DNS Spoofing – redirecting users to fake websites

## Network Security Techniques
Encryption (SSL/TLS)
VPN (Virtual Private Network)
Intrusion Detection System (IDS)
Secure configuration of devices

## Conclusion

Networking is the foundation of cybersecurity. Understanding how networks work helps in identifying vulnerabilities and protecting systems from attacks.
