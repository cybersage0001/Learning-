# Bandit0
--> Bandit0 starts with the SSH server to connect with our virtual machine to share files and perform all upcoming tasks. </br>

* So, let's start with the first step: open Kali(If you don't know how to download Kali, check out the kali/installation folder)    with the command line interface (command terminal) </br>
* After that, write the command to connect: "ssh bandit0@bandit.labs.overthewire.org -p 2220"
* You connected with bandit0 level.
