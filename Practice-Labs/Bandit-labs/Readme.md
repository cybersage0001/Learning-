# Bandit Labs Writeups

This repository contains my personal walkthroughs and solutions for the Bandit wargame.

The purpose of these write-ups is to document my learning journey in Linux, networking, and cybersecurity fundamentals through hands-on practice.

## About Bandit

Bandit is designed for beginners who are new to Linux commands, SSH, file handling, permissions, encoding, networking, and basic security concepts.

Each level introduces a new concept and requires practical problem-solving skills to retrieve the password for the next level.

## Skills Practiced

- Linux Commands
- SSH Authentication
- File Permissions
- Searching and Filtering Files
- Encoding and Decoding
- Networking Basics
- Scripting Fundamentals
- Cybersecurity Problem Solving

## Goal of This Repository

- Track my cybersecurity learning progress
- Improve Linux command-line skills
- Practice real-world troubleshooting
- Build a strong cybersecurity foundation
- Share solutions and learning notes
