Let's practice for the first lab, Bandit1 </br>
First, we need to connect our machine to our attack machine through SSH.
--> open command terminal in kali, write out the command: "ssh bandit1@bandit.labs.overthewire.org -p 2220"

If you have a question about -p 2220, it means the SSH service is running on port 2220, as specified by the banditlabs owners. Otherwise, the SSH default port number is 22.
